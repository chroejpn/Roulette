# カスタムルーレット (Custom Roulette)

ブラウザ上で動く，誰でも簡単に使えるカスタマイズ可能なルーレットWebアプリです．

## 特徴 (Features)
* **項目の自由な編集**: テキストエリアに改行区切りで入力するだけで，すぐにルーレットの盤面が更新されます．
* **プロファイル保存機能**: 作成したルーレットの項目を「プロファイル」としてお使いのブラウザに保存し，いつでも読み込み・削除が可能です．
* **自動カラー生成**: 項目の数に合わせて，ルーレットの区画の色が自動的に生成されます．見やすい区切り線付きです．
* **インストール不要**: Webサイトにアクセスするだけで，スマートフォンやパソコンのブラウザからすぐに利用できます．

## 使い方 (Usage)
1. 公開URLにアクセスするか，ダウンロードした `index.html` をブラウザで開きます．
2. 画面のテキストエリアに，ルーレットに設定したい項目を改行で区切って入力し，「ルーレットを更新」ボタンを押します．
3. 「回す！」ボタンを押すとルーレットがスタートします．
4. 現在の項目リストを保存したい場合は，プロファイル名を入力して「現在の項目を保存」ボタンを押してください．次回以降，ドロップダウンリストから簡単に呼び出すことができます．

# Custom Roulette

A customizable roulette web application that runs in the browser and is easy for anyone to use.

## Features
* **Customizable Items**: Simply enter your items in the text area separated by newlines to instantly update the roulette wheel.
* **Profile Saving Function**: Save your created roulette items as a "profile" in your browser, allowing you to load or delete them at any time.
* **Automatic Color Generation**: The colors of the roulette sections are automatically generated based on the number of items. It comes with clear separator lines.
* **No Installation Required**: Simply access the website to use it immediately from the browser on your smartphone or PC.

## How to Use
1. Access the published URL or open the downloaded `index.html` in your web browser.
2. Enter the items you want to set on the roulette in the text area, separated by newlines, and press the "Update Roulette" button.
3. Press the "Spin!" button to start the roulette.
4. If you want to save the current list of items, enter a profile name and press the "Save Current Items" button. You can easily recall it from the dropdown list next time and onwards.
